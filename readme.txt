1. Клонируйте репозиторий
2. Откройте папку formal-languages
3. Перейдите в ветку l-lexer
4. Выполните команду в командной строке:
	gradlew.bat jar
5. Теперь можно запускать программу:
	java -jar build/libs/lexer-1.0-SNAPSHOT.jar "Полный путь к файлу с кодом на языке L"
6. Результаты выводятся в консоль (стандартный поток вывода)
